# 什么是 Python？

Python 就是一门编程的语言而已。

## 为什么选择 Python？

### 1. 简单易学 📚

比其他语言简单很多，还有很多现成的第三方库，整体来说代码比较精简。

比如完成同一个任务，C 语言要写 1000 行代码，Java 只需要写 100 行，而 Python 可能只要 20 行。

### 2. 用途广泛 🌍

Python 可以用来做很多事情：

| 领域 | 用途 | 
|------|------|
| 🌐 网站开发 | 搭建网站和 Web 应用 | 
| 📊 数据分析 | 处理和分析数据 | 
| 🤖 人工智能 | 机器学习、深度学习 | 
| 🔧 自动化 | 自动处理重复工作 |
| 🎮 游戏开发 | 制作简单游戏 |



## Python 能做什么？

大部分都能做，但最多的还是用来写脚本、数据处理、机器学习等

## Python 缺点

- 运行速度慢
- 代码不能加密

不过这都不重要，能完成我们所需要的功能就行



## 学习 Python 需要什么基础？

**不需要任何编程基础！** 只要你：

- ✅ 会用电脑
- ✅ 认识英文字母
- ✅ 有学习的热情

就可以开始学习 Python 了！
